# 🚀 Быстрый старт Fear Protection Dashboard

## ✅ РЕШЕНИЕ ПРОБЛЕМЫ CORS

Для работы с API нужен прокси-сервер. Следуйте инструкциям ниже:

---

## 📦 Установка и запуск (2 шага)

### Шаг 1: Запустите прокси-сервер

**Терминал 1:**
```bash
node proxy-server.js
```

Вы увидите:
```
=== Fear Protection CORS Proxy Server ===
Server running on http://localhost:3000
```

### Шаг 2: Запустите веб-сервер

**Терминал 2 (новое окно):**
```bash
python -m http.server 8000
```

### Шаг 3: Откройте браузер

Откройте: **http://localhost:8000**

---

## 🎯 Что вы увидите:

✅ **Реальные данные** с Fear Project API  
✅ **Список игроков** с серверов  
✅ **Steam профили** с возрастом аккаунтов  
✅ **VAC баны**  
✅ **Последние репорты**

---

## 🔍 Проверка работы:

1. Откройте консоль браузера (F12)
2. Посмотрите логи:
   - `[App] Application initialized` - приложение запущено
   - `[App] Fetching data...` - загрузка данных
   - `[App] Extracted Steam IDs: X` - найдено игроков
   - `[App] Data fetch completed` - данные загружены

---

## ⚠️ Если не работает:

### Проблема: "Failed to fetch"
**Решение:** Убедитесь, что прокси-сервер запущен (Терминал 1)

### Проблема: "Cannot GET /"
**Решение:** Убедитесь, что веб-сервер запущен (Терминал 2)

### Проблема: "No Steam IDs found"
**Решение:** Проверьте, что на серверах есть игроки

---

## 📊 Архитектура:

```
Браузер (localhost:8000)
    ↓
Прокси-сервер (localhost:3000)
    ↓
Fear Project API (api.fearproject.ru)
Steam API (api.steampowered.com)
```

---

## 🛠️ Альтернативный запуск:

### С помощью npm:
```bash
npm start
```

### Без Node.js (с расширением браузера):
1. Установите "CORS Unblock" для Chrome
2. Включите расширение
3. Откройте index.html напрямую

---

## 📝 Технические детали:

- **Прокси-сервер:** Node.js HTTP сервер на порту 3000
- **Веб-сервер:** Python HTTP сервер на порту 8000
- **Fear API:** `/api/fear/*` → `https://api.fearproject.ru/*`
- **Steam API:** `/api/steam/*` → `https://api.steampowered.com/*`

---

## 🎨 Функционал:

- **Автообновление**: каждые 30 секунд
- **3 колонки**: Новореги, VAC Баны, Репорты
- **Статистика**: количество серверов и игроков
- **Адаптивный дизайн**: работает на мобильных
- **Футер**: кнопки открываются в новых вкладках
